# RailEase_IRCTC_Redesign_prototype

## 📌 Project Overview
RailEase is an interactive UI/UX prototype redesign of the IRCTC railway booking platform created using Figma. The prototype was designed to improve usability, navigation flow, and overall user experience through modern interface design principles.

---

## 🎯 Key Features
- Clickable interactive prototype
- Smooth screen navigation
- Smart Animate transitions
- User-friendly booking flow
- Modern railway booking interface
- Responsive layout structure

---

## 🖥 Screens Included
1. Home Page
2. Search Results Page
3. Train Details Page
4. Passenger Details Page
5. Payment Page
6. Booking Success Page
7. My Bookings Page
